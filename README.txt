Рылов Денис Сергеевич 2016.

Было выполнено задание для математика (http://www.syntacore.com/files/trial_task_math.pdf).

Решение реализовано на языке C++ в виде класса Task. При создании объекта типа Task можно вызвать конструктор с двумя параметрами - именами входного и выходного файла. Конструктор кидает ошибки.

В входном файле содержатся порождающие вектора. Их число, обозначаемое через K, не должно превышать 60, а длина, обозначаемая через N, должна составлять не более 90 символов (нулей и единиц). Все вектора должны быть одной длины и разделены символами новой строки (\r\n или \n). 

Метод loadAllVectors() загружает все вектора из файла в массив. Выполняется за O(N)+O(K).

Метод computeSpectrumInParallel() вычисляет спектр. Принимает на вход один параметр, обозначенном через I, - количество сумм и их весов, вычисляемых параллельно. 0<I<2^{K}. Метод выполняется за O(2^{K}/I)+O(I), если считать, что запись веса в массив выполняется мгновенно. Метод кидает ошибки.

Метод writeSpectrumIntoOutputFile() записывает спектр в выходной файл. Метод выполняется за O(N).

Исходные коды программы должны быть скомпилированы GCC с поддержкой стандарта С++11 (флаг -std=c++11).

Программа может быть оптимизирована следующими способами:
- убрать программные лимиты на K и N;
- реализовать автоматическое определения необходимости распаралеливания в методе loadAllVectors();
- реализовать автоматическое определение I.
